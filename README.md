# ScrollerDemo
Unity Demo: 
An Efficient, Recyclable ScrollerView on UGUI,

Only one script [ScrollerBase.cs],

Support horizontal and vertical mode.

一个高效可复用的ScrollView组件，循环利用GameObject，使列表不会因子物体太多导致卡顿。

目前支持水平，垂直模式，但仅支持子物体尺寸相等的情况。
